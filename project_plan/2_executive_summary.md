## 🎯 2. Executive Summary

### Core Objective
The Marine Biologist Expert agent serves as a specialized assistant for marine biology and oceanography inquiries. It provides users with direct, official resources from the National Oceanic and Atmospheric Administration (NOAA) by exposing a simple tool to retrieve the official NOAA website link.

### High-Level Success Criteria
*   Correctly identifies user queries requesting NOAA or official oceanographic resources.
*   Successfully invokes the custom tool `get_noaa_link` to retrieve the NOAA URL.
*   Presents the URL cleanly across Chat UI, SMS, and Voice interfaces.
